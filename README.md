# PHP Calculator Application

This repository contains a simple calculator web application implemented in PHP.

## Overview

The application allows users to:

- enter two values
- choose an operator
- calculate a result
- receive an error message for invalid input

This PHP version is used as the **source application** for comparison with the Ur/Web translation.

## Security Focus

The PHP calculator was analysed for:

- reflected XSS risk
- parameter tampering attack surface
- division-by-zero handling

## Files

- `calculator.php` – main PHP calculator file

## Run

open:

http://localhost/calculator.php
